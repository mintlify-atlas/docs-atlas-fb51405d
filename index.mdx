Whether you're self-hosting or contributing, this documentation covers everything from spinning up the dev environment to every REST API endpoint.

## What's Inside

HealtyHelp ships as a monorepo with a **React/Vite** frontend and an **Express + MongoDB** backend. Here's what the platform offers:

## Get Started in 4 Steps

<Steps>
  <Step title="Clone and install dependencies">
    Clone the repository and install packages for both the server and the client.
  </Step>
  <Step title="Configure environment variables">
    Copy the `.env` templates and fill in your MongoDB URI, JWT secret, Groq API key, and Google OAuth credentials.
  </Step>
  <Step title="Start the development servers">
    Run `npm run dev` in both `server/` and `client/` to launch the API on port 5000 and the frontend on port 5173.
  </Step>
  <Step title="Create your Super Admin account">
    Run the `initSuperAdmin` script to bootstrap the first admin user, then log in at `/login`.
  </Step>
</Steps>

<Note>
  The full step-by-step setup guide with all commands is on the [Quickstart](/quickstart) page.
</Note>
